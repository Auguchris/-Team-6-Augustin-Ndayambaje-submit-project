 # Project title
  Modeling and Spatial Analysis of Telecom Service Quality (QoS) in Rwanda

## Project Overview
This project analyzes telecom service quality (QoS) in Rwanda using data science, machine learning, and spatial analysis techniques. The study focuses on evaluating network performance indicators such as latency, packet loss, throughput, and signal strength to assess user experience and identify geographical disparities across districts.


## Key Components
- Data cleaning and preprocessing scripts are located in `notebooks/data_cleaning.py`
- Exploratory data analysis and Machine learning models are implemented is in `notebooks/EDA.py`
- Dashboard  of power BI is located in the `notebook/ final report /augustinndayamabaje.pbix`
